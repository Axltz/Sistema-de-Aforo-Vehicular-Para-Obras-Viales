# Sistema-Inteligente-de-Transporte-aplicado-a-la-movilidad-vial

Diseño e implementación de un sistema de visión artificial para el conteo y monitorio vehicular en zonas de obra vial.

## Proposito
Este proyecto tiene como propósito el análisis y diseño de un sistema automatizado de aforo vehicular enfocado en zonas de obra vial, donde actualmente el conteo de vehículos suele realizarse de manera manual por personal en campo.

Estas condiciones justifican la necesidad de explorar soluciones automatizadas que permitan mejorar la precisión y continuidad del monitoreo.

---

## Objetivo General

Analizar, diseñar y proponer la arquitectura de un sistema automatizado de aforo vehicular que pueda implementarse en zonas de obra vial para mejorar la recolección y análisis de datos de tránsito.

---

## Objetivos 

Diseñar conceptualmente un sistema que permita:

- Detectar vehículos mediante video.
- Contabilizar el flujo vehicular automáticamente.
- Generar datos para análisis del tráfico en zonas de construcción.
- Reducir errores asociados al conteo manual.
